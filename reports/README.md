# Published digest archives (GitHub Pages)

Interactive HTML + JSON for each daily run. Open **`index.html`** in a browser for the archive frame (heatmap, year pills, inline digest viewer).

After `python run.py`, new `{prefix}.json` / `{prefix}.html` pairs are written here and `index.html` is rebuilt.

**GitHub Pages:** site root redirects here → `reports/index.html`
