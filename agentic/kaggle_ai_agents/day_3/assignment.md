## What You Will Learn

The whitepaper covers dynamic context management and avoiding context rot by using portable Agent Skills organized around a central `SKILL.md` file. It explains progressive disclosure so prompts stay lightweight while detailed execution logic and tools load only on demand.

In the codelabs, you will:

1. learn how skills work in Antigravity
2. install and use Agents CLI skills
3. create agents, lint code, and test agents with natural language workflows
