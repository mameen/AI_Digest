# Project Spec (Day 1)

## Purpose
Deliver one concise daily AI brief that helps a user track important AI news quickly.

## Functional Requirements

1. ingest stories from curated sources
2. normalize into a common schema
3. deduplicate near-identical stories
4. produce one brief with 8 to 12 cards
5. include source links for each card

## Non-Functional Requirements

1. local run under 2 minutes
2. deterministic validation before output
3. no secrets in repository artifacts
