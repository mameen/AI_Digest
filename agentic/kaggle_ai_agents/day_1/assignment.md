## What You Will Learn

The whitepaper covers the transition from manual syntax coding to intent-driven vibe coding and disciplined agentic engineering. It explains how AI agents compress parts of the traditional software development life cycle (SDLC) and introduces the factory model, where developers design the evaluation, constraint, and context harnesses that safely guide autonomous execution.

The codelabs introduce Antigravity 2.0, the Antigravity IDE, and the Antigravity CLI to build initial applications. They also use Google AI Studio and Cloud Run so the app can be deployed and shared.
