## What You Will Learn

The whitepaper reframes security and evaluation for non-deterministic AI workflows by emphasizing continuous effective trust through a seven-pillar architecture. It highlights safeguards such as:

1. ephemeral sandboxing
2. defenses against hallucinated slopsquatting packages
3. a Red/Blue/Green security triad
4. OpenTelemetry trajectory evaluation

In the codelabs, you will:

1. build a human-in-the-loop agent where manual intervention is possible
2. use ADK, Agents CLI, and Antigravity in one lifecycle
3. practice secure, test-driven agentic development patterns
